AR/VR

  • VR Room
    My very first exploration into virtual reality development using Unity. This interactive 3D environment is designed for VR headsets and allows users to experience an immersive room with natural interaction through hand tracking and VR controls. It began as a practice project from the Unity Learn VR Development pathway and is still in progress.
    GitHub: VR-Room

  • Backgammon VR
    A virtual reality backgammon game for two players in a shared immersive environment. It is being built for 6DoF headsets and includes grab interactions, dice physics, and socket-based movement. This is a personal project currently under development.
    GitHub: VR-Backgammon

  • AURaPath — Augmented Reality Waypoint Determination for UR10 Robots
    An end-to-end augmented reality system for spatial waypoint determination, digital-twin preview, and execution on a physical Universal Robots UR10 collaborative robot using Microsoft HoloLens 2. This project was completed as part of work at the Autonomous Intelligent Robotic Center (AIRC) at the University of Windsor.
    GitHub: WayPointCreator

  • GreenHouse Demo
    A professional prototype developed for greenhouse visualization and monitoring in an immersive environment. This project is private, so only limited public details are shown here.
